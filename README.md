SYS Property Holdings Real Estate Management System

Description
This is a web-based Online-to-Offline (O2O) real estate management platform designed for SYS Property Holdings. The system serves strictly as a lead generation and property viewing appointment platform. 
It allows customers to browse property catalogs, calculate estimated monthly installments, book offline showroom visits, and apply for affordable housing schemes (including an automated lucky draw feature). 

Note: All financial transactions and contract signings are handled offline at physical showrooms. Which mean this website dont support any PAYMENT features
